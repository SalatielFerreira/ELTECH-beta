# 📱 Como Instalar o ELTECH no Celular

## Método 1: Netlify (MAIS FÁCIL - 2 minutos)

1. Acesse: **https://app.netlify.com/drop**
2. Arraste a pasta **eltech-pwa** inteira para a página
3. Pronto! Vai gerar um link tipo: `https://seu-site.netlify.app`
4. Abra esse link no celular
5. O app vai mostrar um botão **"Instalar ELTECH"**
6. Toque nele → o app aparece na tela inicial como um app normal!

> ✅ Gratuito para sempre, sem cartão de crédito

---

## Método 2: GitHub Pages (Também grátis)

1. Crie uma conta em **https://github.com** (se não tiver)
2. Crie um repositório novo chamado **eltech**
3. Faça upload de TODOS os arquivos da pasta **eltech-pwa**
4. Vá em Settings → Pages → Source: **main** → Save
5. Aguarde 1-2 minutos
6. Seu app estará em: `https://seuusuario.github.io/eltech`
7. Abra no celular e instale!

---

## Como instalar no celular (após hospedar):

### Android (Chrome):
- Abra o link no Chrome
- Vai aparecer automaticamente o banner "Instalar ELTECH"
- Ou toque no menu (⋮) → "Instalar aplicativo"

### iPhone (Safari):
- Abra o link no Safari
- Toque no botão compartilhar (↑)
- Toque em "Adicionar à Tela de Início"
- Toque em "Adicionar"

---

## Arquivos incluídos:
- `index.html` → O app completo
- `manifest.json` → Configuração do app (nome, ícone, cor)
- `sw.js` → Service Worker (funciona offline)
- `icon-192.png` → Ícone pequeno
- `icon-512.png` → Ícone grande
- `COMO-INSTALAR.md` → Este arquivo

## Funciona offline?
SIM! Depois de instalado, o app funciona mesmo sem internet.
Os dados ficam salvos no seu celular.
